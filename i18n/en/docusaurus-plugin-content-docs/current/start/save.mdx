## Save Version Version History {#version}

Saving a version allows you to return to a specific point in time on the canvas.
By saving versions, you can navigate to the state of the canvas at any desired point using the version history.

### Save Version {#version-save}

1. To save a version, click `Save Version Control> Save Version`.

    <img src={save02} width="200px" alt="ë²„ì „ì €ìž¥ ì´ë¯¸ì§€" />
