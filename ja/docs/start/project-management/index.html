完成したプロジェクト管理


完成したプロジェクトは修正·コピー·削除できます。


プロジェクトの詳細情報を修正する​

プロジェクト名を修正する​

  1. プロジェクトの詳細情報を修正するためにテンプレートメニューで修正を希望するプロジェクトをクリックします。
  2. ポップアップが表示されたらプロジェクト名に編集アイコンをクリックします。
  3. プロジェクト名を修正します。
  4. もう一度編集アイコンをクリックして修正を完了します。

 

プロジェクトの説明を修正する​

  1. プロジェクトの詳細を修正するためにテンプレートメニューで修正を希望するプロジェクトをクリックします。
  2. ポップアップが表示されたらプロジェクトの詳細に編集アイコンをクリックします。
  3. プロジェクトの詳細を修正します。
  4. もう一度編集アイコンをクリックして修正を完了します。

 

プロジェクトメンバーの追加及び削除​

プロジェクトメンバーの追加及び削除についてはこちらを参考にしてください。

 

 

プロジェクトのコピー及び削除​

プロジェクトのコピー​

  1. プロジェクトをコピーするためにテンプレートメニューで修正を希望するプロジェクトの三点リーダーをクリックします。
  2. コピーをクリックします。
  3. ポップアップが表示されたらプロジェクトをコピーするフォルダーを選択します。
  4. プロジェクト名を作成します。
  • 英語、ハングル、数字のみ許可
  • 2 ~ 20 桁
  1. プロジェクトの説明を作成します。
  • 100 文字以内
  1. コピーするボタンをクリックします。

 

プロジェクトの削除​

  1. プロジェクトを削除するためにテンプレートメニューで削除を希望するプロジェクトの三点リーダーをクリックします。
  2. 削除をクリックします。
  3. ポップアップが表示されたらプロジェクト名をもう一度入力します。
  4. 削除するボタンをクリックします。

 

 

キャンバスの修正​

  1. キャンバスを修正するためにテンプレートから修正を希望するプロジェクトをクリックします。
  2. ポップアップが表示されたら右側のプロジェクトを修正するボタンをクリックします。
