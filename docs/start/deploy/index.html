배포관리

last update: 2024.09.01

클라우드 스튜디오는 단순히 시각적인 인프라 아키텍처를 제공하는 것을 넘어서, 캔버스에 그린 인프라를 실제 CSP에 배포할 수 있는 기능을 제공합니다. 이를 통해 사용자들은 쉽게 클라우드 인프라를 구축할 수 있습니다.

 

배포 점검​

IaC의 문법적 오류나 설정이 올바르게 되었는지 가상으로 검증하는 단계입니다. 실제 CSP에 배포되지 않습니다. (--dry-run option과 같은 역할을 합니다.)

  1. 배포테스트를 진행하기 위해서 배포관리 > 유효성 검사를 클릭합니다.

  1. 좌측 메뉴에서 유효성 검사하기 버튼을 클릭합니다.
  2. 자격증명을 선택합니다.
  3. 유효성 검사하기 버튼을 클릭합니다.

 

정보

배포테스트를 진행하기 전 자격증명을 필수적으로 등록해야 합니다. 자격증명을 등록하는 방법은 여기 를 참고해주세요.

 

 

배포이력​

배포이력은 실제 배포된 인프라에 대한 이력을 보여줍니다.

  1. 배포 이력을 확인하기 위해서 배포 > 배포이력을 클릭합니다.

  1. 최근 배포 이력을 확인합니다. 이력을 삭제하려면 원하는 이력을 선택한 후 삭제하기 버튼을 클릭합니다.
