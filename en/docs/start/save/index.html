There are two options to save the canvas.

  • Save
  • Save Version

Save saves the current state of the canvas. Saving the canvas is similar to saving a project. The image you see on the project thumbnail will be the canvas image after saving.


Saving is simple. Click File > Save.

Saving a version allows you to return to a specific point in time on the canvas. By saving versions, you can navigate to the state of the canvas at any desired point using the version history.

Save Versionâ€‹

  1. To save a version, click Save Version Control> Save Version.

  1. Enter a Tag Message. The tag message you provide will appear as the Version Title in the version history.


  1. Click Save.

Version Historyâ€‹

  1. To check the version history, click Version Control > Version History.

View the saved versions in the Version History.

note
  • Latest: Recent Canvas
  • Manual: Version save history
  • Planed: Deploy Validation Check
  • Applied: Deployment completed history

Click on the canvas you want to revert to.
