Deployment Management


CloudStudio goes beyond merely providing a visual infrastructure architecture by offering functionality to deploy the infrastructure drawn on the canvas to a CSP.


This allows users to easily build cloud infrastructure.

 

Deployment Review​

This is the stage where the syntax errors and configurations of IaC are virtually validated to ensure correctness. The actual deployment to a CSP does not occur at this stage.

  1. To perform a deployment test, click on Management > Deploy Validation Check.

  1. From the left menu, click the "Run Deploy Validation Check" button.
  2. Select Credentials.
  3. Click the Run Deploy Validation Check button.

 

info

Before proceeding with the deployment test, you must register credentials as a prerequisite. For information on how to register credentials, please refer to here.

 

 

Deployment History​

Deployment history shows the history of infrastructure that has been actually deployed. You can check the current deployment status and recent deployment history.

  1. To view the deployment history, click on Deploy > Deploy History.

  1. Review the recent deployment history. To delete a history entry, select the desired entry and click the Delete button.
